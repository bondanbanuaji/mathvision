MathVision - Smart Scan-Based Math Calculator

A full-stack web application built with MEVN (MongoDB, Express.js, Vue.js, Node.js) 
that revolutionizes how students solve mathematical problems. Upload or capture 
math problems through your camera, leverage OCR technology to parse equations, 
and receive comprehensive solutions with step-by-step explanations.

Key Features:
✅ OCR-powered math problem scanning
✅ Comprehensive calculator (Algebra, Calculus, Trigonometry, Statistics, Matrix)
✅ Step-by-step solution explanations
✅ Interactive 2D/3D graph plotting
✅ Personal calculation history & analytics
✅ AI-powered learning mode with concept explanations
✅ User authentication & profile management
✅ Dark mode support
✅ Responsive design for all devices

Tech Stack:
- Frontend: Vue.js 3 (Composition API), Pinia, Vue Router, Axios
- Backend: Node.js, Express.js, JWT Authentication
- Database: MongoDB (Mongoose ODM)
- Math Engine: math.js, Tesseract.js (OCR), KaTeX (LaTeX rendering)
- Visualization: Plotly.js / Chart.js

Perfect for students, educators, and anyone who needs quick, reliable 
mathematical solutions with educational value.
